# Zero-Copy research

Здесь можете найти код связанный с моим исследованием подхода zero-copy как замены тривиальных методов копирования данных

## Запуск

Для запуска необходимо в командной строке использовать вписать

```Bash
python3 experiments/experiment{n}.py
```

После запуска должны быть показаны графики и данные полученные в ходе тестирования.
